# Ciberseguridad en América Latina (2018-2024)

[![Mendeley Data](https://img.shields.io/badge/Mendeley%20Data-ctw6xndgwc-red)](https://data.mendeley.com/drafts/ctw6xndgwc) [![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/) [![Archivos](https://img.shields.io/badge/Archivos-20-blue)]()

## Descripción
Datos sobre ciberataques, ransomware e infraestructura digital en América Latina (2018-2024). Fuentes: ITU GCI 2020/2024, Kaspersky Security Network, IBM X-Force, IDB/OEA, CERT nacionales.

**Mendeley Data:** [ctw6xndgwc](https://data.mendeley.com/drafts/ctw6xndgwc) | **EN version:** [cybersecurity-latin-america](https://github.com/juanmoisesd/cybersecurity-latin-america)

## Estadísticas Clave (2024)
| Indicador | Valor | Fuente |
|-----------|-------|--------|
| Detecciones LATAM (2024) | 164M+ (5 países mayores) | Kaspersky |
| Crecimiento ransomware 2018-2024 | +324% | IBM X-Force |
| Pérdidas financieras estimadas (2024) | USD 6.2 mil millones | ITU/IDB |
| Brecha de profesionales | 193,000 vacantes | ISC2/ISACA |
| País más afectado | Brasil (35% incidentes) | IBM X-Force |
| Caso emblemático | Costa Rica emergencia nacional 2022 | CISA |
